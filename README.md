# phicomn-k3-weather
用于斐讯K3 [k3screenctrl]([](https://github.com/lwz322/k3screenctrl)) 的天气更新脚本





## 简介

相比于其他脚本，改进并支持了以下特性：

- 支持显示错误信息，方便定位
- LuCI配置更新后立即刷新
- 网络错误30秒后自动重试
- 天气信息及IP定位仅使用彩云API




![Screen](/preview.png)




## 安装

上传并替换到以下路径：

```shell
/lib/k3screenctrl/weather.sh
```
